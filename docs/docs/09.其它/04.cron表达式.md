## cron 表达式

- Media Saber 中有较多涉及设置时间周期的采用`cron表达式`进行设置。

### 表达式含义

表达式有五位 分别对应如下：

```
* * * * *
分 时 天 月 周
```

- `周`要么直接星号，要么具体设置只支持英文格式，如：`MON/TUE/WED/THU/FRI/SAT/SUN`。

### 举例子

```
0 8 * * * 每一天的8点执行任务

0 */8 * * * 每间隔8小时执行任务

0 0-8 * * * 每一天0-8点每小时执行一次任务

0 0-8/2 * * * 每一天0-8天每2小时执行一次任务

0 1,6,8 * * * 每一天1点\6点和8点执行一次任务


```
