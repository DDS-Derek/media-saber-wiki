# 👤 我的信息

"我的信息"是 MediaSaber 系统中用于管理用户个人资料和账户安全设置的功能模块。通过该功能，用户可以查看和修改个人信息、更改密码、管理 API 密钥以及控制设备登录状态。

## 🚪 功能入口

在我的信息功能位于系统首页顶部导航栏右侧，通过点击用户头像可以打开下拉菜单：

在下拉菜单中点击"我的信息"选项，即可进入我的信息配置页面。

## 🖥️ 界面布局

我的信息页面采用标签页形式组织内容，分为"账户信息"和"安全配置"两个主要部分：

### 📋 账户信息标签页

账户信息标签页包含用户的基本信息和头像设置功能：

#### 🖼️ 修改头像

在该区域可以修改用户的个人头像：

1. 显示当前用户头像
2. 点击"头像设置"按钮打开头像设置对话框
3. 在对话框中可以上传新头像或选择系统提供的默认头像
4. 确认后新头像将立即生效

#### 🔐 修改密码

用户可以在此修改账户登录密码：

##### 密码修改步骤：

1. **输入当前密码**：在"当前密码"输入框中输入现有密码
2. **设置新密码**：在"新密码"输入框中输入想要设置的新密码
3. **确认新密码**：在"确认密码"输入框中再次输入新密码
4. **保存修改**：点击"保存修改"按钮完成密码更改

##### 密码要求：

- 新密码必须与确认密码一致
- 密码应具有足够的复杂度以确保安全
- 修改成功后需要使用新密码登录

#### 📝 基本信息

用户可以在此修改个人基本信息：

##### 可修改信息：

- **昵称**：用户的显示名称

##### 修改步骤：

1. 在"昵称"输入框中输入新的昵称
2. 点击"保存修改"按钮
3. 修改成功后系统会显示确认信息

### 🔒 安全配置标签页

安全配置标签页包含账户的高级安全设置：

#### 🔑 API KEY 管理

用于管理外部设备和工具调用 MediaSaber API 的密钥：

##### 使用方法：

> 💡 **使用建议**：具体使用方式请参考 [API鉴权文档](/docs/other/api_auth/) 说明，包含请求头、查询参数等不同场景的使用规范

> ⚠️ **安全提示**：请勿在日志或公共配置中明文存储API KEY

| 操作 | 功能说明 |
|------|----------|
| ➕ 新增 | 创建新的 API KEY |
| 👁️ 显示/隐藏 | 切换 API KEY 的可见性 |
| ⏸️ 启用/停用 | 控制 API KEY 的使用状态 |
| 🗑️ 删除 | 删除指定的 API KEY |

##### 使用场景：

- **MCP服务认证**：用于第三方MCP客户端连接Media Saber的MCP服务，详见 [MCP服务文档](/docs/ai/mcp/)
- **Webhook集成**：媒体服务器（Emby/Jellyfin）通过Webhook与Media Saber通信时的身份验证，详见 [Webhook文档](/docs/other/webhook)
- **微信消息通知**：企业微信接收Media Saber消息通知时的API认证，详见 [微信文档](/docs/other/weixin)
- **API调用**：外部工具调用Media Saber API时的身份验证
- **自动化脚本**：脚本或程序访问系统功能时使用

#### 📱 设备管理

用于管理当前账户登录的设备：

##### 功能特点：

- 显示所有已登录的设备列表
- 标识当前正在使用的设备
- 显示设备类型、IP 地址和登录时间
- 支持单独注销设备或批量注销

##### 操作功能：

| 操作 | 功能说明 |
|------|----------|
| 🚫 注销设备 | 注销指定设备的登录状态 |
| 🚫 注销所有设备 | 一次性注销除当前设备外的所有设备 |

##### 设备标识：

- 🖥️ 桌面设备（web）
- 📱 iOS 设备（ios）
- 🤖 Android 设备（android）
- 💻 其他设备（laptop）

## 📋 详细操作指南

### 🖼️ 头像设置操作流程

1. 进入"我的信息"页面
2. 在"账户信息"标签页中找到头像区域
3. 点击"头像设置"按钮
4. 在弹出的对话框中选择或上传新头像
5. 确认选择后关闭对话框
6. 新头像立即生效

### 🔐 密码修改操作流程

1. 进入"我的信息"页面
2. 在"账户信息"标签页中找到"修改密码"区域
3. 输入当前密码
4. 输入新密码并确认
5. 点击"保存修改"按钮
6. 系统验证通过后密码修改成功
7. 下次登录时需使用新密码

### 📝 基本信息修改流程

1. 进入"我的信息"页面
2. 在"账户信息"标签页中找到"基本信息"区域
3. 修改昵称等信息
4. 点击"保存修改"按钮
5. 系统保存修改并显示成功提示

### 🔑 API KEY 管理操作流程

#### 新增 API KEY

1. 进入"我的信息"页面
2. 切换到"安全配置"标签页
3. 在"API KEY"区域点击"新增"按钮
4. 系统生成新的 API KEY 并显示在列表中

#### 显示/隐藏 API KEY

1. 在 API KEY 列表中找到目标 KEY
2. 点击眼睛图标切换可见性
3. 显示时可以看到完整 KEY 值
4. 隐藏时显示为星号

#### 启用/停用 API KEY

1. 在 API KEY 列表中找到目标 KEY
2. 点击播放/暂停图标切换状态
3. 启用状态可以正常使用
4. 停用状态无法使用

#### 删除 API KEY

1. 在 API KEY 列表中找到目标 KEY
2. 点击垃圾桶图标
3. 确认删除操作
4. KEY 从列表中移除且无法恢复

#### 使用 API KEY

获取API KEY后，根据不同的使用场景选择合适的认证方式：

**请求头认证方式**：
- 适用场景：MCP服务、大多数API调用
- 配置方法：在HTTP请求头中添加 `apiKey: your-api-key`

**查询参数认证方式**：
- 适用场景：Webhook回调、微信消息通知
- 配置方法：在URL后添加 `?apiKey=your-api-key`

> ⚠️ **安全提醒**：无论使用哪种方式，都要确保API KEY的安全性，避免在日志中暴露完整的KEY值。

### 📱 设备管理操作流程

#### 注销指定设备

1. 进入"我的信息"页面
2. 切换到"安全配置"标签页
3. 在"设备信息"区域找到目标设备
4. 点击设备右侧的垃圾桶图标
5. 确认注销操作
6. 该设备将被强制退出登录

#### 注销所有设备

1. 进入"我的信息"页面
2. 切换到"安全配置"标签页
3. 在"设备信息"区域点击"注销所有设备"按钮
4. 确认操作
5. 除当前设备外的所有设备都将被注销

## ⚠️ 注意事项

### 🔐 安全建议

1. **密码安全**：
   - 定期更换密码
   - 使用复杂密码（包含大小写字母、数字和特殊字符）
   - 不要在多个网站使用相同密码

2. **API KEY 管理**：
   - 妥善保管 API KEY，避免在代码或日志中明文存储
   - 定期更换 API KEY 以确保安全
   - 删除不再使用的 KEY
   - 根据使用场景选择合适的认证方式（请求头或查询参数）
   - 在生产环境中优先使用请求头方式，避免KEY在URL中暴露

3. **设备管理**：
   - 定期检查登录设备列表
   - 发现异常设备立即注销
   - 在公共设备使用后及时注销

### 🔄 操作提示

1. **信息保存**：
   - 修改信息后记得点击"保存修改"按钮
   - 系统会验证输入内容的有效性

2. **操作确认**：
   - 删除和注销操作都需要二次确认
   - 请谨慎执行不可逆操作

3. **界面切换**：
   - 使用标签页切换不同功能模块
   - 当前选中标签页会高亮显示

通过"我的信息"功能，用户可以全面管理个人账户信息和安全设置，确保账户安全并个性化使用体验。
