# 📊 业务日志

业务日志是 MediaSaber 系统中记录业务操作和事件的重要工具。与系统日志不同，业务日志专注于记录用户操作、业务流程和业务相关事件，帮助管理员了解系统业务运行情况和用户行为。

## 🚪 功能入口

业务日志位于系统首页顶部导航栏右侧，通过点击带有搜索图标的文档图标可以进入业务日志页面：

点击图标后，系统会跳转到业务日志页面。

## 🖥️ 界面布局

业务日志页面采用列表形式展示，包含筛选、操作和详情查看功能：

### 🔧 顶部操作栏

页面顶部包含各种操作和筛选功能：

#### ✅ 全选功能

- 勾选全选框可以选择当前页面显示的所有日志
- 支持批量操作
- 当部分日志被选中时显示为中间状态

#### 🔍 日志筛选

提供两种筛选方式：

1. **标题搜索**：
   - 根据日志标题关键字进行搜索
   - 支持模糊匹配
   - 实时搜索，输入即筛选

2. **类型筛选**：
   - 根据日志类型进行筛选
   - 下拉选择需要查看的日志类型
   - 支持多选

#### 🗑️ 批量删除

- 选择多个日志后，点击删除按钮可以批量删除
- 删除前会弹出确认对话框
- 提高日志管理效率

#### 🔄 刷新按钮

- 点击刷新按钮可以重新加载日志列表
- 获取最新的日志信息
- 更新显示内容

### 📋 日志列表区域

页面中部是日志列表展示区域：

#### 📝 日志项结构

每个日志项包含以下信息：

- **复选框**：用于选择该条日志
- **类型描述**：显示日志的业务类型
- **标题**：日志的简要描述
- **创建时间**：日志记录的时间
- **删除按钮**：删除该条日志

#### 📂 日志详情查看

- 点击日志项可以展开查看详细信息
- 展开后显示日志的完整内容
- 支持折叠操作

### 📄 分页控件

页面底部包含分页控件：

- 显示当前页码和总页数
- 支持跳转到首页、末页、上一页、下一页
- 可以选择每页显示的日志数量

## 📋 日志操作

### 📂 查看日志详情

1. 点击任意日志项
2. 日志项会展开显示详细内容
3. 再次点击可折叠详情内容

### 🗑️ 删除单条日志

1. 点击日志项右侧的删除按钮（垃圾桶图标）
2. 系统会弹出确认对话框
3. 确认后该条日志会被删除

### ✅ 批量操作

#### 批量选择

1. 勾选需要操作的日志项前的复选框
2. 或者勾选顶部全选框选择所有日志

#### 批量删除

1. 选择多个日志项
2. 点击顶部的批量删除按钮
3. 确认删除操作
4. 选中的日志将被批量删除

### 🔍 日志筛选

#### 标题筛选

1. 在标题搜索框中输入关键字
2. 系统会自动筛选匹配的日志
3. 清空搜索框可重置筛选

#### 类型筛选

1. 点击类型筛选下拉框
2. 选择需要查看的日志类型
3. 系统会筛选出相应类型的日志

### 🔄 刷新日志

1. 点击刷新按钮
2. 系统会重新加载最新的日志列表
3. 更新显示内容

## 📊 日志类型

业务日志支持多种类型，便于分类管理：

### 🎬 媒体相关

- 媒体搜索日志
- 媒体下载日志
- 媒体整理日志

### 🌐 站点相关

- 站点签到日志
- 站点数据同步日志
- 站点连接测试日志

### 📺 订阅相关

- 订阅创建日志
- 订阅更新日志
- 订阅删除日志

### ⚙️ 系统操作

- 配置更新日志
- 系统维护日志
- 用户操作日志

## 📋 日志详情页面

点击日志项可以进入日志详情页面，查看更多详细信息：

### 🎯 页面功能

#### 📋 日志基本信息

- 显示日志标题
- 显示日志类型
- 显示创建时间

#### 🔍 详细内容搜索

- 提供关键字搜索功能
- 在当前日志详细内容中搜索
- 快速定位关键信息

#### 🏷️ 日志级别筛选

支持按日志级别筛选：

| 级别 | 说明 | 按钮颜色 |
|------|------|---------|
| Info | 一般信息 | 蓝色 |
| Debug | 调试信息 | 紫色 |
| Error | 错误信息 | 红色 |

### 📜 日志内容显示

- 以滚动列表形式展示日志详细内容
- 显示每条日志的时间戳
- 根据日志级别使用不同颜色标识

### 📄 分页功能

- 支持大量日志内容的分页显示
- 可以选择每页显示的条目数
- 支持跳转到指定页面

### 🔙 返回功能

- 点击返回按钮可返回业务日志列表页面
- 保持操作连贯性

## 🎯 使用技巧

### 高效日志管理

1. **定期清理**：定期删除过期或不必要的日志
2. **分类查看**：使用类型筛选功能查看特定业务日志
3. **批量操作**：使用批量删除功能提高管理效率

### 问题诊断

1. **时间定位**：根据问题发生时间筛选相关日志
2. **关键字搜索**：使用关键字快速定位问题信息
3. **级别筛选**：重点关注Error级别日志

### 日志分析

1. **趋势分析**：通过日志时间分布分析业务趋势
2. **频率统计**：统计各类操作的执行频率
3. **异常检测**：通过日志模式识别异常行为

通过业务日志功能，管理员可以全面了解系统业务运行情况，追踪用户操作，诊断业务问题，为系统优化和业务决策提供数据支持。
