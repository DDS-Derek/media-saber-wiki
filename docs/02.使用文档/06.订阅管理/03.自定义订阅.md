# ⚙️ 自定义订阅

## 📋 功能说明

自定义订阅功能允许用户创建更灵活的订阅规则，适用于特定的订阅需求。通过自定义订阅，您可以：

- 创建基于关键字的订阅规则
- 设置复杂的过滤条件
- 定制下载和保存策略
- 管理特殊的站点配置

## 🎯 使用场景

自定义订阅适用于以下场景：

- 订阅特定关键字的资源
- 创建复杂的筛选规则
- 管理非标准的媒体资源
- 设置特殊的下载策略
