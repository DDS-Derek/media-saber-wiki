## 1、说明

media saber 不提供任何站点资源服务，但支持网友自行适配站点配置文件。 适配很简单，只需要一个文本编辑器+浏览器即可适配。 技术栈也仅仅只需要会 HTML+CSS 基础知识，理解 CSS 选择器即可。

此站点配置适用于主流私有站点和简单公开站点的索引。如需其他复杂公开站点索引采集请看：[深度采集](/docs/sites/deep_collection)

## 2、配置文件说明

文件名必须为站点域名，域名之间的点替换为英文下划线。比如www.baidu.com，配置文件的文件名为www_baidu_com.json

## 3、配置文件说明

站点配置文件均为 json，需要遵守 json 基本语法。
