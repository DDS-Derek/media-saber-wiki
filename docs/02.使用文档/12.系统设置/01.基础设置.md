# 🎛️ 基础设置

> 本文档详细介绍 Media Saber 系统各项基础配置。每个字段均配有用途说明、默认值、示例和操作建议，便于快速上手和查阅。

---

## 1️⃣ 基础配置

| 字段名             | 说明                                                                 | 默认值/示例                | 操作建议/帮助                         |
|--------------------|----------------------------------------------------------------------|----------------------------|----------------------------------------|
| **代理服务器**     | 将使用代理访服务器访问themoviedb、telegram、google等境外网站和服务，站点默认不使用代理，如需使用需在站点维护中开启；配置格式示例：http://127.0.0.1:7890（Http协议）、socks5://127.0.0.1:8018、socks5h://127.0.0.1:8018(remote DNS) | 空<br>`http://127.0.0.1:7890`<br>`socks5://127.0.0.1:8018` | 如需访问 TMDB、Telegram、程序更新等境外服务，需配置代理。 |
| **外网访问地址**   | 用于外网访问访问本系统使用的地址，通常为http协议://域名:端口，如：http://www.example.com:8888 | 空<br>`http://your.domain:8443` | 填写完整协议和端口，Telegram Webhook 推荐 88 或 8443。 |
| **内网访问地址**   | 用于内网访问访问本系统使用的地址 | 空<br>`http://192.168.1.100:8888` | 通知等场景可用内网地址。               |
| **登录后跳转页面** | 可以指定登录跳转指定页面菜单，不再是默认首页 | `/dashboard`                | 可选填，如 `/dashboard`、`/media`。选项包括：默认首页、我的媒体库、寻影探剧、搜索任务、站点数据、站点签到、站点资源、站点公告、站点消息、电影订阅、电视剧订阅、正在下载、目录同步、小工具 |
| **User-Agent**     | 程序所有对外请求都将默认使用此User-Agent。如发现被豆瓣、PT站等封堵，请适当修改此项 | 当前浏览器 UA               | 如被豆瓣/PT站封禁可自定义 UA。可点击输入框后的箭头按钮获取当前浏览器的User-Agent。 |
| **存储空间目录**   | 需要统计存储空间的目录, 多个以英文逗号分隔 | 空<br>`/mnt/media,/mnt/downloads` | 用于空间统计展示。                    |
| **文件管理目录**   | 文件管理目录, 多个以英文逗号分隔 | 空<br>`/mnt/downloads`      | 用于文件管理页面操作。                 |
| **临时文件保留天数** | /app/config/tmp 路径下的文件 | `3`                         | 定期清理临时文件，避免空间占用。单位为天数。 |

## 2️⃣ 媒体配置

| 字段名             | 说明                                                                 | 默认值/示例                | 操作建议/帮助                         |
|--------------------|----------------------------------------------------------------------|----------------------------|----------------------------------------|
| **TMDB API Key**   | 在themoviedb.org网站申请API Key，该项必须配置，否则所有功能无法正常运行 | 空<br>`xxxxxxx`            | 无此项无法正常匹配媒体。必须配置此项才能使用系统功能。 |
| **TMDB 接口地址**  | TMDB接口地址，可自建代理服务，留空默认使用TMDB官方地址。第三方代理域名地址不保证完全可用，请斟酌使用。 | `https://api.themoviedb.org/3` | 官方无法访问时可填写代理地址。可选官方地址：https://api.themoviedb.org/3 或 https://api.tmdb.org/3 |
| **TMDB 语言**      | TMDB语言，将会影响媒体信息展示、文件整理及刮削等所有涉及使用TMDB数据的内容语言 | `zh-CN`                    | 如需英文信息可填 `en-US`。             |
| **TMDB 匹配模式**  | tmdb匹配模式.严格模式:严格匹配标题、媒体年份和类型  正常模式：若识别的年份和类型未获取到，则去掉年份和类型，根据名称再查一次进行匹配 | `正常模式`                  | 推荐正常模式，特殊需求可选严格。选项包括：正常模式、严格模式 |
| **TMDB 电影缓存天数** | 缓存tmdb，提升系统流畅度。新媒体可能信息有误，缓存后可能更新不及时。 | `7`                        | 缓存过期后自动刷新。                   |
| **TMDB 电视剧缓存天数** | 缓存tmdb，提升系统流畅度。新媒体可能信息有误，缓存后可能更新不及时。电视剧建议稍微短一点 | `1`                        | 缓存过期后自动刷新。                   |
| **TMDB 超时时间(秒)** | 请求超时时间, 单位秒，建议不超过30秒 | `60`                       | 网络慢时可适当调高。                   |
| **TMDB 索引保留天数** | 为 0 则永久保留 | `0`                        | 索引用于加速搜索。                     |
| **TMDB Web 搜索**  | tmdb api 搜索不到时尝试 tmdb web 搜索(例如: 2099 2099 y:2025, 这样搜索如果只有一条记录则匹配成功) | 关闭                       | 极端场景下提升识别率，耗时增加。        |
| **媒体搜索结果包含成人内容条目** | 刮削/检索时包含 TMDB 成人内容。                        | 关闭                       | 如需成人内容请开启。                   |
| **媒体种子精准搜索** | 搜索时匹配 TMDB 精准过滤种子。                                   | 关闭                       | 提升搜索准确率。                       |
| **手动搜索只搜 BT 站点** | 手动搜索时只搜索 BT 站点。                                   | 关闭                       | 如只需 BT 资源可开启。                 |

## 3️⃣ 订阅搜索

| 字段名             | 说明                                                                 | 默认值/示例                | 操作建议/帮助                         |
|--------------------|----------------------------------------------------------------------|----------------------------|----------------------------------------|
| **媒体搜索结果包含成人内容条目** | 刮削/检索时包含 TMDB 成人内容。                        | 关闭                       | 如需成人内容请开启。                   |
| **媒体种子精准搜索** | 搜索时匹配 TMDB 精准过滤种子。                                   | 关闭                       | 提升搜索准确率。                       |
| **手动搜索只搜 BT 站点** | 手动搜索时只搜索 BT 站点。                                   | 关闭                       | 如只需 BT 资源可开启。                 |
| **BT 资源排序优先** | BT 资源排序优先于其他资源。                                       | 关闭                       | 下载优先选择 BT 资源。                 |
| **BT资源走网盘**    | BT 资源走网盘下载。                                               | 关闭                       | 如需自动转存网盘可开启。               |
| **媒体库缺失**      | 订阅时只订阅媒体库缺失的内容。                                    | 关闭                       | 避免重复订阅已入库内容。               |
| **资源搜索默认过滤规则** | 搜索种子时默认使用的过滤规则。                               | 空<br>`1080p|x265|国语`    | 支持正则表达式。                       |
| **种子排序规则**    | 搜索种子时默认排序规则。                                          | `做种数优先`                | 影响下载资源选择。                     |
| **站点超时时间(秒)** | 站点相关请求超时时间。                                           | `10`                       | 网络慢时可适当调高。                   |
| **站点下载超时时间(秒)** | 下载请求超时时间。                                            | `30`                       | 下载慢时可适当调高。                   |
| **订阅刷新周期**    | RSS 订阅刷新的时间间隔（秒）。                                   | `600`                      | 最小 300 秒。                          |
| **订阅搜索周期**    | 定时检索订阅资源（小时）。                                       | `6`                        | 最小 6 小时。                          |
| **搜索前缀**        | 消息渠道端搜索媒体和种子的前缀。                                  | `搜索`                     | 可自定义命令前缀。                     |
| **订阅前缀**        | 消息渠道端订阅媒体的前缀。                                        | `订阅`                     | 可自定义命令前缀。                     |
| **种子RSS缓存时间(分)** | RSS 结果缓存时间。                                            | `10`                       | 缓存过期后自动刷新。                   |
| **种子搜索缓存时间(分)** | 搜索结果缓存时间。                                            | `10`                       | 缓存过期后自动刷新。                   |
| **搜索任务保留天数** | 搜索任务保留天数。                                               | `3`                        | 定期清理历史任务。                     |

## 4️⃣ 媒体整理

| 字段名             | 说明                                                                 | 默认值/示例                | 操作建议/帮助                         |
|--------------------|----------------------------------------------------------------------|----------------------------|----------------------------------------|
| **电影重命名格式** | 定义电影文件/目录命名格式，支持占位符。                             | `{title} ({year})/{title} ({year}){ext}` | 点击前端“?”按钮查看详细说明。           |
| **电视剧重命名格式** | 定义电视剧文件/目录命名格式，支持占位符。                         | `{title} ({year})/Season {season}/{title} - S{season}E{episode}{ext}` | 点击前端“?”按钮查看详细说明。           |
| **整理默认转移方式** | 文件整理转移方式。                                                  | `硬链接`                    | 推荐硬链接，节省空间。                 |
| **整理最小文件大小(M)** | 整理时忽略小于此大小的文件。                                    | `50`                        | 单位 MB，避免转移小文件。              |
| **整理硬连接后触发变化** | 针对部分设备硬连接后文件不刷新的问题。                          | 关闭                        | 极空间等设备建议开启。                 |
| **整理路径忽略**   | 整理时忽略匹配路径的文件，支持正则。                               | 空<br>`/sample/`            | 路径包含指定内容则忽略。               |
| **整理文件名忽略** | 整理时忽略匹配文件名的文件，支持正则。                             | 空<br>`trailer|sample`      | 文件名包含指定内容则忽略。             |
| **查补漏集排除媒体库** | 查漏集、补漏集时排除的媒体库名称。                              | 空<br>`动画,纪录片`          | 多个用英文逗号分隔。                   |
| **整理高质量文件覆盖** | 整理时高质量文件覆盖原文件。                                    | 关闭                        | 同名文件按体积覆盖。                   |
| **过滤规则详细日志** | 临时查找过滤问题用，严禁长时间开启。                              | 关闭                        | 仅排查问题时开启。                     |
| **识别词日志**     | 识别词日志，通常较多。                                             | 关闭                        | 仅排查问题时开启。                     |
| **识别日志**       | 识别日志，通常较多。                                               | 关闭                        | 仅排查问题时开启。                     |
| **已入库效果**     | 已入库的媒体弱化显示。                                             | 开启                        | 视觉区分已入库内容。                   |
